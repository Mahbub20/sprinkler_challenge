# sprinkler_challenge
Calculate number of sprinklers can be fitted in the room and caulculate coordinates of each sprinkler.

## Prerequisites
- .NET SDK 7.x or 8.x (installed and on PATH)
- Git

## Quick setup
1. Clone
   git clone <repo-url>
   cd "sprinkler_challenge"

2. Build
   dotnet build

3. Run (to see the output like the Output.txt file)
   dotnet run

4. There is an sample Output.txt file where we can see the output of the program after sucessfully running the program.
